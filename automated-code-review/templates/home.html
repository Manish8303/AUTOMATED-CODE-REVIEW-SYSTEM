About the Project

This project, proudly developed at Chandigarh University, Punjab, India, is designed to revolutionize code quality and accelerate software releases. We've created an intelligent, automated solution that goes beyond traditional code checkers to provide actionable, context-aware insights for every developer.

How It Works: A Smart, Two-Step Process

Our system combines the meticulousness of static analysis with the intelligence of machine learning to deliver comprehensive and accurate code reviews.

Step 1: The Code Detective 🔍

Our journey begins with a thorough Static Analysis of your codebase. Without even running the program, our system acts as a meticulous detective, scanning every line against a comprehensive set of rules and best practices. We leverage SonarQube to find common issues like:

  • Bugs & Errors: Catching potential crashes and logical flaws.
  • Vulnerabilities: Identifying security risks such as SQL injection or unsecured APIs.
  • Code Smells: Pointing out areas that are difficult to read or maintain.

This initial step provides a solid foundation for identifying rule-based, deterministic problems quickly and efficiently.

Step 2: The Intelligent Mentor 🧠

This is where we go beyond traditional tools. After the static analysis, our custom Machine Learning (ML) models take over. Trained on a massive dataset of real-world open-source projects from GitHub, our models understand code context and patterns. This allows them to:

  • Provide Smarter Recommendations: Suggesting elegant and efficient solutions that a simple rule-based system would miss.
  • Reduce False Positives: Distinguishing a true bug from a harmless coding pattern, reducing "alert fatigue" for developers.
  • Predict Future Issues: Identifying code structures that are historically prone to bugs, helping you fix them before they become a problem.

By learning from real-world code, our system provides a nuanced, intelligent review that adapts and improves over time.
